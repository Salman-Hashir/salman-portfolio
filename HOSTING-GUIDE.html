Update Your Links First
Open index.html in Notepad / VS Code and replace these placeholders
1
Open index.html in a text editor
Right-click index.html → Open with → Notepad (Windows) or TextEdit (Mac). Or use VS Code if you have it.
2
Find and replace your LinkedIn URL
Press Ctrl+F (Find) and search for this text:
Find this:
https://linkedin.com/in/YOUR-USERNAME

Replace with your real LinkedIn URL, example:
https://linkedin.com/in/salman-hashir
3
Find and replace your GitHub URL
Find this:
https://github.com/YOUR-USERNAME

Replace with your real GitHub URL, example:
https://github.com/salman-hashir
4
Add project live links (when ready)
When you deploy your projects, replace the placeholder project URLs:
AnonConnect — find and replace:
https://YOUR-ANONCONNECT-URL.vercel.app

Vaartha — find and replace:
https://YOUR-VAARTHA-URL.vercel.app

Niggasuran — find and replace:
https://YOUR-NIGGASURAN-URL.vercel.app
Tip
Until you deploy those projects, the buttons still appear — visitors just can't open them yet. That's fine.
5
Save the file
Press Ctrl+S to save index.html after making changes.
03
Deploy on Vercel
Free hosting — your site goes live in under 2 minutes
100% Free
No coding needed


1
Go to vercel.com
Open your browser and go to vercel.com — click Sign Up and create a free account (use your GitHub or Google account).
2
Click "Add New" → "Project"
Once logged in, you'll see your dashboard. Click the button that says Add New at the top right, then click Project.
3
Choose "Deploy without a Git repository"
You'll see options. Scroll down and look for a section that says "Deploy without Git" or click the button that says "Browse" to upload files directly.
ℹ Vercel has a drag-and-drop uploader. You don't need GitHub for this.
4
Drag your entire project folder
Drag the entire salman-portfolio folder into the Vercel upload area. Vercel will detect it's a static site automatically.
⚠ Do NOT upload the HOSTING-GUIDE.html file. Only upload index.html, books/ folder, and resume/ folder.
5
Click "Deploy"
Vercel will process your files (takes about 30–60 seconds). You'll see a progress bar.
6
Your site is live!
Vercel will give you a URL like:
salman-hashir.vercel.app
✓ Your portfolio is now live on the internet. Share this URL on your CV, LinkedIn, and WhatsApp.

05
How to Link Your Projects
So visitors can click and see your working apps
1
Deploy each project separately on Vercel
Each of your projects (AnonConnect, Vaartha, Niggasuran) needs to be deployed separately. For each project folder:
Option A — GitHub (recommended):
1. Push your project code to GitHub
2. In Vercel → Import from GitHub
3. Select the repo → Deploy
4. Get URL like: anonconnect.vercel.app

Option B — Direct upload:
1. Zip your project folder
2. Drag to Vercel → Deploy
3. Get URL like: anonconnect-xyz.vercel.app
2
Copy the live URL and paste in index.html
Once each project is deployed, copy its Vercel URL and replace the placeholder in index.html (as shown in Phase 2, Step 4). Re-upload index.html to your portfolio on Vercel.
✓ Now when visitors click "View Live Demo" on your portfolio, they'll go directly to your working project!
06
How Books Work
Read Online and Download buttons explained
What each button does
👁 Read Online → Opens the PDF directly in the browser. Visitor can scroll and read without downloading. Works for Anantharam & Eryndor (PDF files).
⬇ Download → Saves the file to the visitor's device. Works for all 5 books.
Coming Soon → Shown for Kaalavritham since it's ongoing. Change this when you finish it.
ℹ The .docx books (Dark Worship, Paranormal Gaveshanam, Bodhavum Abodavum) are download-only since browsers can't display Word files inline. If you want Read Online for these too, convert them to PDF using Word → Save As → PDF, then replace the .docx files with .pdf versions and update the links in index.html.
